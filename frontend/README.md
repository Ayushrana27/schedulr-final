# 🚀 Schedulr – Calendly Clone

A full-stack scheduling platform inspired by Calendly that allows users to create events, set availability, and book meetings seamlessly.

---

## ✨ Features

### 📅 Event Management
- Create event types (title, duration, slug)
- Edit existing events
- Delete events
- Unique public booking links

### ⏰ Availability System
- Set availability by day of week
- Define start and end time
- Dynamic slot generation based on availability
- Timezone support (basic)

### 📌 Booking System
- Calendar-based date selection
- Dynamic slot generation based on:
  - Availability
  - Event duration
- Prevent double booking
- Booking confirmation page

### 📊 Meetings Dashboard
- View upcoming meetings
- View past meetings
- Cancel meetings

### 🎨 UI/UX
- Clean modern UI inspired by Calendly
- Fully responsive (mobile + desktop)
- Smooth user experience

---

## 🛠 Tech Stack

### Frontend
- React.js
- Axios
- React Router

### Backend
- Node.js
- Express.js

### Database
- PostgreSQL (via Prisma ORM)
